## SWG: Vanguard — Ship Concepts & EWAR Documentation

This repository contains the complete ship design concepts and Electronic Warfare (EWAR) documentation I created (and helped refine with the team) for the **SWG: Vanguard** private server.

I worked alongside the core development team on these systems during active development. The documents cover everything from hull stats, weapon hardpoints, and flight mechanics to advanced EWAR modules, debuff stacking, countermeasure logic, and balance tuning.

**Permission was explicitly granted by the SWG: Vanguard project leads and team** to publicly display and host this documentation as part of my portfolio. All content remains the intellectual property of the Vanguard team and SWG IP holders; it is shared here solely to showcase my contributions to game design, systems documentation, and collaborative development on a live private server.

This is **not** official Star Wars Galaxies material — it is fan-created content developed for the Vanguard community and server.
